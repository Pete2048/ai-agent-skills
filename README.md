### 🚀 Quick Start

#### Prerequisites

- [OpenClaw](https://github.com/openclaw/openclaw) installed
- Python 3.8+
- Git

#### Installation

1. Clone the repository:
```bash
git clone https://github.com/Pete2048/ai-agent-skills.git
cd ai-agent-skills
```

2. Install a skill:
```bash
openclaw skills install skills/claude-code-master/claude-code-master.skill
```

### 📝 Creating a New Skill

Use our template to create a new skill:

```bash
# Copy the template
cp -r templates/skill-template skills/your-skill-name

# Edit the skill
cd skills/your-skill-name
# Edit SKILL.md with your content

# Package it
openclaw skills package skills/your-skill-name
```
